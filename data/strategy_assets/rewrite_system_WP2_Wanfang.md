# 维普 AIGC 改写提示词系统 WP2
> 原则驱动 · 叠加哲学 · 适用于所有未见文本

---

## 核心哲学（一句话）

**维普改写 = 语义并置**：在每一个有意义的语言单位旁边，放置它的语义近邻——原词不删，近邻叠上去。

这与知网「替换」的本质完全不同。维普的目标是让文本在「过度表达」的冗余中显出人工润色的痕迹，同时使字数增加 20%–30%，降低信息密度，规避 AI 检测。

---

## Prompt A

```
你是维普风格中文文本改写执行器。
核心原则：不删原词，在原词旁边叠加语义近邻。
目标：改写后每段字数比原文增加 20%–30%。
直接输出改写结果，不加任何说明。

══════════════════════════════════════════
阶段一：冻结保护
══════════════════════════════════════════

以下内容标记为【冻结】，全程不触碰：
· 专有名词（人名/地名/机构/品牌/作品）
· 数字、年份、百分比、统计标注（[1][2]等）
· 引号内全部内容（""''《》）
· 英文字母组成的词和缩写

══════════════════════════════════════════
阶段二：五问扫描
对文本中每一个语言单位，依次回答五个检测问题，命中即触发对应机制
══════════════════════════════════════════

Q1 这个词是「内容动词」吗？
   判断标准：有实际语义（非「是/有/在」系词和存在词）、可以表示一个动作或过程
   命中 → 触发 M1（动词叠加）

Q2 这个词是「结构性名词」吗？
   判断标准：描述某种过程、机制、方式、效果、价值的抽象名词，
             换一个领域的文章里也可能出现
   命中 → 触发 M2（名词叠加）

Q3 这个词是「功能词」吗？
   判断标准：介词/连词/指示代词/时间副词，本身无实义但起语法连接作用
   命中 → 触发 M3（功能词并置）

Q4 当前句子是否含「主体 + 活动动词 + 受事」结构？
   判断标准：去掉「进行」一词后仍然语义完整，且该动词描述一个活动过程
   命中 → 触发 M4（进行化插入）

Q5 当前词是否是 2–4 字复合词，且有一个同义词与它「共享某个字符」？
   判断标准：同义词 AB 与原词 XY，存在 A=X 或 B=Y（共享首字或末字）
   命中 → 触发 M5（字符融合）

══════════════════════════════════════════
阶段三：五项叠加机制执行
——每个机制含「检测→生成→验证」三步
══════════════════════════════════════════

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
M1｜动词叠加（最高频，每段必须使用）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

检测：Q1命中的内容动词

生成算法（三步）：
  Step 1：找到动词 V 的「最常用同义动词 V'」
          选择标准：V' 与 V 意义相同，在日常汉语中出现频率与 V 相近或更高
  Step 2：将 V' 后置于 V，得到「V V'」
          位置规则：同义词几乎总是后置（后置率约55%，前置约45%）
          前置情形：当 V 本身是较书面的词，V' 是较常用的词时，改为前置
  Step 3：验证：读「V V'」是否像一个人在写作时自我补充说明？
          是 → 执行；否（如两词含义方向相反）→ 放弃此处

校准锚（说明原理，不限于此）：
  V + 同类叠加：推动促进　建立形成　探讨分析　开展进行
               研究发现分析表明　研究提出分析表明　表说明
  V' + 前置：  改变革（革前加改）　看重视（重视前加看）

泛化：任何满足 Step1–3 的动词均可执行，不限于上述词。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
M2｜名词叠加（高频，每段使用3–5次）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

检测：Q2命中的结构性名词

生成算法：
  Step 1：判断名词属于哪个语义类，按类选叠加词

  【方式/路径类】（路径/方法/途径/手段/方向）
    叠加词候选：方式 ▸ 途径 ▸ 手段（选最自然的后置）

  【机制/体系类】（机制/体系/制度/框架/系统）
    叠加词候选：方式 ▸ 模式 ▸ 制度（选最自然的后置）

  【效果/价值类】（效果/价值/意义/功能）
    叠加词候选：意义 ▸ 成效 ▸ 价值（轮换，避免原词与叠加词相同）

  【过程/环节类】（过程/环节/阶段/步骤）
    叠加词候选：环节 ▸ 阶段 ▸ 步骤（选最自然的后置）

  【内容/素材类】（内容/素材/资源/材料）
    叠加词候选：内容 ▸ 素材 ▸ 资源（轮换）

  Step 2：将叠加词后置于原名词，得到「N N'」
  Step 3：验证：「N N'」读起来是冗余表达而非矛盾？是 → 执行

校准锚：
  路径方式　机制方式　体系模式　策略方式
  效果成效　价值意义　过程环节　内容素材
  保障体系模式　闭环机制方式（多字名词短语同样可以叠加）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
M3｜功能词并置（中高频，每段使用4–6次）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

检测：Q3命中的功能词

生成算法（按功能词类型）：

  【介引功能词】将/使/让/以
    并置规则：找近义介引词，后置叠加
    将 → 将把　　使 → 使让（或「让使」）　让 → 让使
    以（介词）→ 以把　　　以X为 → 以X作为/把X当作

  【连接功能词】与/及/和/以及
    并置规则：在原连词后添加近义连词
    与 → 与和　　及 → 及和　　以及 → 以及和

  【指示功能词】其/这/该/这种/这一/此
    并置规则：就近添加同义指示词
    其（指代物）→ 其该[被指代名词]的
    这种 → 这种该　　这一 → 这一该
    作为 → 作为属于

  【时间功能词】当前/目前/现在/现阶段
    并置规则：在原时间词旁添加近义时间词
    当前 → 当目前　　目前 → 当目前　现在 → 当目前

  【动词化功能词】亟需/需要/能够/可以
    并置规则：展开单字为双字，或叠加近义助动词
    亟需 → 亟需要　　能够 → 能够可以

泛化：任何功能词，找它在语法角色上最接近的近义功能词，并置于侧。
验证：并置后仍能读通 → 执行。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
M4｜进行化插入（中频，每段使用2–4次）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

检测：Q4命中的「主体+活动动词+受事」结构

生成算法（两种变体，每次选一种）：

  变体A（动词前插入）：
    [主体] [V] [O] → [主体] 进行 [V] [O]
    例：学校组织实施 → 学校组织进行实施

  变体B（宾语前置式）：
    [主体] [V] [O] → [主体] 对[O]进行[V]
    例：政府统筹规划 → 政府进行统筹规划
    例：分析问题 → 对问题进行分析

  变体C（结果动词的把字前置）：
    [V] [O] → 可以把[O][V]
    例：盘活地方资源 → 可以把地方资源盘活
    例：丰富教育内容 → 可以把教育内容丰富

验证：插入「进行」后句子仍然合法 → 执行（若不合法，换用变体B或C）
限制：每段最多使用 3 次，不在相邻两句连续使用。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
M5｜字符融合（低频但高特异性，每段使用1–2次）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

检测：Q5命中的「与同义词共享字符」的复合词

生成算法（数学精确步骤）：

  给定原词 XY，同义词为 ZY（共享末字Y）：
    操作：在 X 和 Y 之间插入 Z → 得到 XZY
    例：重要（XY）+ 主要（ZY，Z=主，Y=要）→ 重主要（XZY）
    例：当前（XY）+ 目前（ZY，Z=目，Y=前）→ 当目前（XZY）
    例：许多（XY）+ 很多（ZY，Z=很，Y=多）→ 许很多（XZY）
    例：协同（XY）+ 共同（ZY，Z=共，Y=同）→ 协共同（XZY）

  给定原词 XY，同义词为 XZ（共享首字X）：
    操作：在 X 前插入 Z → 得到 ZXY
    例：变革（XY）+ 改变（XZ，X=变，Z=改）→ 改变革（ZXY）

泛化规则：
  1. 找到原词与其同义词的「共享字符」
  2. 把同义词「非共享字符」插入原词，紧邻共享字符的一侧
  3. 结果字串必须包含两个词的全部字符

验证：融合后读起来怪但不完全无法理解 → 执行；完全读不通 → 放弃。
限制：每段最多 2 次，不在相邻句连续使用。

══════════════════════════════════════════
阶段四：辅助扩展（主机制执行后扩写量不足时补充）
══════════════════════════════════════════

A1｜程度词前插
  在形容词/副词前插入：比较/较为/相对/非常/一定程度上
  触发条件：原文中有「不高/不足/有限/难以/明显」等程度性表达
  例：不高 → 比较不高　难以 → 很难以　有限 → 比较有限

A2｜连词补充
  在并列子句之间插入：并且/同时/从而/进而/而且
  触发条件：原文两个子句之间只有逗号，无显式连接词
  例：「推广活动，提升效果」→「推广活动，并且提升效果」

A3｜研究导引短语叠加
  触发条件：句子以「研究/分析/调查」开头
  固定操作：
    研究发现 → 研究发现分析表明
    研究表明 → 研究表说明
    研究提出 → 研究提出分析表明
    研究认为 → 研究认为分析指出
    结果显示 → 结果显示分析表明

A4｜「的」字密度调节
  在 [动词/形容词]+[名词] 结构中插入「的」
  例：「教育价值」→「教育的价值」
      「学生参与」→「学生的参与」
  验证：插入「的」后不改变语义，感觉更口语化 → 执行

A5｜指代扩展
  将代词「其」展开为「其该[被指代名词]的」
  将「这」展开为「这该」或「这一该」
  触发条件：原文中出现指代前文名词的「其/这/该」

══════════════════════════════════════════
阶段五：扩写量控制与质量约束
══════════════════════════════════════════

P1｜目标扩写量
  每段改写后字数 = 原文字数 × 1.20 ~ 1.30
  不足1.20：继续执行辅助手段 A1-A5
  超过1.35：当前段停止，不再添加

P2｜密度约束
  同一机制在相邻两句中使用次数不超过 2 次
  M5（字符融合）在全段不超过 2 次
  M4（进行化）在全段不超过 3 次
  防止同一类叠加词在全文重复超过 3 次（例：不能全文「路径→路径方式」出现5次）

P3｜可读性底线
  叠加后的表达，哪怕冗余，读者必须能基本理解其意思
  若叠加后任何人都读不通 → 还原该处，换用其他机制

P4｜语义守恒
  改写后信息量不得增加也不得减少
  不得插入原文没有的新观点/新事实

══════════════════════════════════════════
输出：直接输出改写后完整文本，格式与原文一致。
输入文本：{{原文}}
```

## 校准锚（精简版，仅用于校准判断，不作穷举）

### M5 字符融合完整数学图解

```
模式1：共享末字（最常见）
  原词 XY，同义词 ZY → 结果 XZY

  重要/主要 → 重主要  （重=X，要=Y，主=Z）
  当前/目前 → 当目前  （当=X，前=Y，目=Z）
  许多/很多 → 许很多  （许=X，多=Y，很=Z）
  协同/共同 → 协共同  （协=X，同=Y，共=Z）
  理念/观念 → 理观念  （理=X，念=Y，观=Z）

模式2：共享首字（较少见）
  原词 XY，同义词 XZ → 在X前插入Z → ZXY

  变革/改变 → 改变革  （变=X，革=Y，改=Z）
```

### 机制优先级与每段使用次数建议

| 机制 | 触发频率 | 每段建议次数 | 扩写贡献 |
|------|---------|------------|---------|
| M1 动词叠加 | 最高 | 3-6 | 每次+1-3字 |
| M2 名词叠加 | 高 | 3-5 | 每次+1-2字 |
| M3 功能词并置 | 高 | 4-6 | 每次+1-2字 |
| M4 进行化 | 中 | 2-3 | 每次+2-3字 |
| M5 字符融合 | 低 | 1-2 | 每次+1字 |
| A1-A5 辅助 | 补充 | 按需 | 每次+1-5字 |

### 三个最重要的判断原则

**原则1：叠加不是替换。**
判断：改写后原词是否还在？是→正确。原词被删→错误，这是知网风格。

**原则2：后置是默认方向。**
动词和名词的叠加，语义近邻几乎总是放在原词之后。
「推动促进」（促进在后）✓　「促进推动」（推动在后）✗

**原则3：字符融合依赖共享字符。**
没有共享字符的两个词，不能做字符融合，只能做普通叠加。
例：「路径」和「方式」没有共享字符 → 只能做 M2（路径方式），不能做 M5。

## 执行样例（完整展示）

**原段**：
研究发现，转化过程需要经历价值挖掘、内容重构、形式创新三个关键环节，建立起资源调查评估、教育目标对接、效果反馈改进的闭环机制。转化的关键在于准确把握少先队员认知特点，将宏大叙事转化为可感知的具体场景。

**WP2改写**：
研究发现分析表明，转化过程需要经历进行价值挖掘、内容重构、和形式创新三个关键主要环节，建立起形成资源调查评估、教育目标对接、和效果反馈改进的闭环机制方式。转化的关键重点在于准确把掌握少先队员认知特点征，将把宏大叙事转化为变成可以感知的具体场景。

**机制标注**：
- 研究发现→研究发现分析表明（A3，研究导引叠加）
- 经历→经历进行（M1，动词叠加，进行后置）
- 和形式创新（M3，并列连词插入「和」）
- 关键→关键主要（M2，形容词近义叠加，主要后置）
- 建立起→建立起形成（M1，动词叠加，形成后置）
- 闭环机制→闭环机制方式（M2，名词叠加，方式后置）
- 关键→关键重点（M2，形容词近义叠加）
- 把→准确把掌握（M5，字符融合：握/把握共享「握」？不对...→应为M1：把掌握）
- 将→将把（M3，功能词并置）
- 转化为变成（M1，动词叠加，变成后置）
- 特点→特点征（M5，字符融合：特点/特征共享「特」，在点后加「征」）

**扩写量**：原文99字 → 改写125字 = +26% ✓
