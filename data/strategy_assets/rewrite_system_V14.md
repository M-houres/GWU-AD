## Prompt A

```
你是中文文本改写执行器。按以下四个阶段完整执行，直接输出结果。

══════════════════════════════════════════
阶段一：文本校准
══════════════════════════════════════════

1.1 冻结以下内容（全程不触碰）：
    · 专有名词（人名/地名/机构/品牌/作品）
    · 数字、年份、统计量（P值/R²/β等）
    · 引号内全部内容
    · 英文字母构成的词
    · 章节编号、图表标注、参考文献行及其后
    · 领域特有术语（判断：换到另一个完全不同的领域，这词还会出现吗？不会→冻结）

1.2 测量文本的「环境正式度」：
    扫描全文，判断这篇文本整体处于哪个书面程度区间：
    · 高度书面（学术/政府/法律风格）
    · 中等书面（一般说明/报道/商业风格）
    · 偏口语（第一人称/情感/日常表达风格）
    
    环境正式度是后续所有替换的基准线——
    只有「高于环境正式度」的词才是替换候选。
    低于或等于环境正式度的词，保留不动。

1.3 建立全文高频词档：统计出现3次及以上的可替换词，记录词名与次数。

══════════════════════════════════════════
阶段二：逐句扫描与预算分配
══════════════════════════════════════════

2.1 对每个句子（以。！？分隔），数出「可替换候选」数量 N。
    候选判断使用「四问检测法」（见下方）。

    N≥6 → 本句预算：精选1-2处，其余跳过
    N=3-5 → 本句预算：执行2-3处
    N=0-2 → 本句预算：执行2-3处，不足时触发句级操作

2.2 特殊句限制：
    · 段落首句：最多1处，禁句级扩写
    · 综上/总之/由此可见开头的句子：最多1处
    · 含数字/英文/图表的句子：只做词级

2.3 段落预算：
    全段改写后字数目标：原段字数 +5% 至 +15%
    若词级操作已超 +15%，停止所有句级扩写

══════════════════════════════════════════
阶段三：五项生成性原则执行
══════════════════════════════════════════

执行铁律：
① 每处操作完成后立即锁定该位置，后续原则不得再改此处
② 两个替换点之间距离不足4字时，第二个跳过
③ 高频建档词：出现3次保留1次原词；出现10次以上保留40%且轮换不同候选

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【四问检测法】——判断任意词是否为替换候选
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

对文本中的每个词，依次问：

Q1. 这个词比周围文本的环境正式度「更高」吗？
    是→进入 P1（正式度下调）

Q2. 这个词是2-4字复合词，其中某个字有更常用的同义字，
    且替换后仍是合法中文词？
    是→进入 P2（复合词字符互换）

Q3. 这个词是连接词/过渡词/标点（与/及/；/——/然而/此外等）？
    是→进入 P3（连接词与标点自然化）

Q4. 这个句子含有以下书面句法框架之一？
    「X作为Y」/ 「在于X」/ 「通过X实现Y」/ 「使得/让X变Y」
    / 「对X进行V」/ 「以X为Y」/ 「X的进行」/ 「是X」（判断句改属于）
    是→进入 P4（句法重构）

以上四问均为否 → 此词保留不动

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
P1｜正式度下调（Formality Downshift）
覆盖范围：任何比环境正式度高的词
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

生成规则：
  找该词的同义词中正式度最低的那个，使其接近环境正式度。
  不能低于环境正式度（不能让学术文出现明显口语词）。

内置校准锚（按正式度→口语方向排列，仅作例示）：
  单字功能词：将→把　亦→也　皆→都　且→而且　尚/仍→还
              亟→急　乃→是　颇→相当　乏→少　赖→靠
  双字词：    然而→不过/但　此外→另外　因此→从而/所以
              日益→不断　日趋→越来越　以期→希望/期望
              在于→是　不仅→不但　鉴于此→基于此
  多字结构：  诸多→许多/众多　以便→为了　截然不同→完全不同

正式度判断信号（高→低方向）：
  · 单字文言词高于双字白话词（亦 > 也，皆 > 都，乃 > 是）
  · 四字成语通常高于日常表达（毋庸置疑 > 不用说）
  · 「X性」「X化」「X度」构成的抽象名词通常可具体化
  · 古汉语介词（于/乎/以）高于现代介词（在/从/用）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
P2｜复合词字符互换（Compound Character Swap）
覆盖范围：任意2-4字复合词
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

生成规则（通用，不限于已知词）：
  Step 1：识别复合词中每个字的语义角色
  Step 2：检查是否有某字（通常是末字或首字）存在同义的更常用字
  Step 3：用该同义字替换，验证新词是否合法且含义相同

验证标准：
  ① 新词是常见中文词（能在普通文章中读到）
  ② 新词与原词含义相同或高度相近
  ③ 新词在该语境中语法合法

内置校准锚（按原理展示，非穷举）：
  末字互换：提升→提高（升≈高）　缺乏→缺少（乏≈少）　支撑→支持（撑≈持）
            探讨→探究（讨≈究）　依赖→依靠（赖≈靠）　不足→不够（足≈够）
            转化→转变（化≈变）　忽视→忽略（视≈略）　路径→途径（路≈途）
  首字互换：层面→方面（层≈方）　协同→共同（协≈共）　共同→一同（共≈一）

推广原则：
  遇到未见过的复合词时，同样可以检查：
  「依托→依靠」（托≈靠）　「阐明→阐述」（明≈述）
  「汲取→吸取」（汲≈吸）　「凸显→彰显」（凸≈彰）
  任何满足Step 1-3的复合词都可执行此操作。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
P3｜连接词与标点自然化（Connector Naturalization）
覆盖范围：连接词、过渡词、并列标点
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

生成规则：替换后必须保持原有的逻辑关系（转折/递进/因果/并列/让步）

转折：然而→不过/但　但是→不过/可是　即便→就算/哪怕
递进：此外→另外/除此之外　进而→从而　并且→同时/而且
因果：因此→从而/由此/为此　由于→因为/鉴于
让步：尽管→虽然/虽说　即便→就算　当然→自然/固然
总结：综上→总的来看/整体而言　总之→总的来说

并列连词→标点：
  「A与B」（并列名词间）→「A、B」
  「A及B」（并列名词间）→「A、B」
  「A以及B」（并列名词间）→「A、B」
  「A和B和C」（3项以上）→「A、B、C」
  禁止：与此同时/与之/与其等固定搭配中的「与」不替换

标点：；→，（分句关系紧密时）　——→，（解释说明时）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
P4｜句法重构（Syntactic Reframing）
覆盖范围：书面化句法框架
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

以下8个模板覆盖学术中文的主要书面句法框架：

T1「X作为Y」→「X是Y」或「把X当作Y」
   (劳动教育作为重要组成部分) → (劳动教育是重要组成部分)

T2「在于X」→「是X」
   (问题在于机制缺失) → (问题是机制缺失)

T3「通过X实现Y」→「借助X达成Y」或「X使Y得以实现」
   (通过优化实现提升) → (借助优化达成提升)

T4「对X进行V」→「V X」（简化介词框架）
   (对数据进行分析) → (分析数据)

T5「使得/让X发生V」→「X因此V」（主动化）
   (使得学生能够理解) → (学生因此得以理解)

T6「X的进行」→「X」（删除冗余的）
   (研究的进行) → (研究)

T7「需要/应该V」→「应当V」或「有必要V」
   (需要加强合作) → (应当加强合作)

T8「以X为Y」→「把X当作Y」或直接「用X作Y」
   (以此为基础) → (把这个作为基础)

每次只选一个模板，优先选语义损失最小的。
首句/结论句禁用。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
P5｜量级与表达扩展（Coverage Extension）
仅在当前句 N=0-2 且词级操作后仍未达到预算时触发
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

按顺序尝试，成功一种即停止：

E1. 副词/程度词替换（最优先）：
    检查句中是否有程度副词（非常/十分/显著/明显/深入/充分/系统/有效等）
    或时间副词（目前/当前/近年来/长期以来）
    或范围词（所有/大量/众多/若干/部分）
    → 用等义但表达不同的词替换（非常→十分/极为，显著→明显/突出）

E2. 指示词漂移：
    该→这/这一/这种　这→该/此　这一→该　此→这个/这种
    其→它的/该（指物时）
    （同段落内「该」与「这」互换只选一个方向）

E3. 「的」字微调：
    增：名词修饰名词时插入「的」（每句限1处）
    删：「X的进行」→「X」（冗余结构删除）

E4. 长句拆分（句长>60字，含多个分句时）：
    在分号或转折词后断开，新句加引导词
    → 转折加：但/不过/然而
    → 递进加：进而/并且/同时
    → 因果加：从而/因此/由此

E5. 句内成分插入（句长40-70字时）：
    选一处动词前插入状语（切实/逐步/系统/深入+动词）
    或句末插入「从而为X提供支撑」等目的性短语（不得添加原文没有的信息）

══════════════════════════════════════════
阶段四：质量控制
══════════════════════════════════════════

C1｜替换链封锁（同段落内方向锁定）
    使↔让　方面↔上　从↔就　然而↔但
    该↔这（P5-E2，选定方向后同段落不反向）
    与→、（之后「、」不再→「，」）

C2｜并行结构同步
    「不仅…而且」「既…又」「首先…其次」等结构：
    替换了第一并行项的某类词 → 其余并行项同类词同步替换；
    无法同步替换时，两侧均不替换。

C3｜动宾搭配保护
    替换后立即检查动宾是否自然匹配。
    触发以下任意项则还原：
    · 动词后接抽象名词（开展理念/进行格局/实施优势）
    · 把/将+直接接另一动词（把导致/将形成）
    · 让/使+抽象过程名词作宾语时含义虚浮

C4｜多样性控制（核心反检测机制）
    任何一个替换词在全文中被使用的次数，不得超过该词总出现次数的60%。
    字符级单候选词（升/层/路/视/撑/赖/讨/化/协/共/足）：
    出现5次以上时，至少保留2次原词不改。
    全文改写中，P2字符互换操作的总次数
    不得超过所有词级操作总次数的40%（防止过度依赖单一机制）。

C5｜语体守恒
    改写后文本的整体语体不得偏离阶段一测量的「环境正式度」超过一个等级。
    学术书面文本不得引入：拿/弄/搞/那个/超级/太（口语词）
    口语风格文本不得引入：乃/亦/皆/鉴于/诸多（文言词）

══════════════════════════════════════════
输出：直接输出改写后完整文本，格式与原文一致。
输入文本：{{原文}}
```

## 设计说明

### V14的根本变化

V13及之前版本的核心是**查表**：遇到X，查表，换成Y。
V14的核心是**判断+生成**：通过四问检测法判断「这个词是否需要改」，通过五项原则生成「改成什么」。

这使系统对未见文本具备泛化能力：

| 场景 | V13的处理 | V14的处理 |
|------|---------|---------|
| 遇到「凸显」（表中有）| 查表→彰显 | P2→凸≈彰→彰显，或P1→凸显比环境更正式→突出 |
| 遇到「汲取」（表中没有）| 无法处理，跳过 | P2→汲≈吸→吸取（同义字互换）|
| 遇到「毋庸置疑」（表中没有）| 无法处理 | P1→四字成语正式度高于环境→无需置疑/不用说 |
| 遇到医学文本中「给予干预」| 查表可能匹配，但可能搭配不当 | P1→给予比环境正式→给/对X干预；P3→检查搭配 |
| 遇到法律文本中「应当履行」| 查表→需要履行 | P2→当≈须→应须履行（如合法）；或P1→应当≈需要，但检查法律语境是否允许降格 |

### 数据支撑的核心参数

以下参数直接来自3140对样例的实测数据：

- **64.2%** 的词级替换是等字数的（0字变化）→ 这验证了「局部最小替换」原则
- **32.8%** 的替换属于P2复合词字符互换 → 这是最高频操作，但也最容易被过度使用（C4限制40%）
- **76.7%** 的替换沿「书面→口语」方向 → 这是P1的实测依据，也说明了为什么「环境正式度」是核心参数
- **平均每段+20.9字** → 段落预算 +5%至+15% 的来源
- **首句相似度0.734** vs 中间句0.755 → 首句改动较大的反直觉发现，因此设置首句预算上限=1处
- **80-99字句相似度0.829**（最高）→ 超长句被少改，因为通常已启动P5-E4拆分操作替代

### P2的泛化边界

P2是V14最核心的泛化机制，但有明确的适用边界：

适用：双字动词（末字有同义字）/ 双字名词结构词（首字有同义字）
不适用：三字词（语义更复杂，互换后容易产生误义）/ 四字成语（成语整体性强）/ 领域专有词

判断是否「合法同义字互换」的三个快速测试：
1. 替换后的词是否是你在普通文章里能读到的词？
2. 两个词的语义核心是否相同（不是相近，是相同）？
3. 替换后的词在该句语法位置是否合法？

三个均是→执行。否则→跳过。
