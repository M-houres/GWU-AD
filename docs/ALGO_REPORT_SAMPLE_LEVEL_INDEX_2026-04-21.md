- `A-triple`
  - 已形成 `原文 + 改后文 + 报告`
  - 可直接进入严格 benchmark 候选池
- `B-pair`
  - 已形成 `原文 + 改后文`
  - 暂无可确认同题报告
  - 可进入弱监督正样本池
- `B-signal`
  - 只有报告
  - 可进入平台信号池
- `C-archive`
  - 压缩包或待抽取文件
  - 需二次处理后再入库
- `C-single`
  - 只有正文或样例
  - 可进入规则素材池
